# Books Store React Application

## Overview

This project is a frontend Books Store application built using React, TypeScript, and Vite. 

The goal of this project was to practice building a structured React application using reusable components, proper state management, and a clean project architecture.

It demonstrates understanding of modern React development using functional components and TypeScript for better type safety.

---

## Features

- Component-based architecture
- Responsive UI layout
- Book listing interface
- Structured project setup using Vite
- TypeScript integration
- ESLint configuration for code quality

---

## Tech Stack

- React 18
- TypeScript
- Vite
- ESLint
